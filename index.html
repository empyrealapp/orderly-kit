A Framework for Orderly Trading Strategies

This site contains the project documentation for the orderly-kit project that is a module for interacting with Orderly Network. It aims is to give you a framework to build trading automation, and backtesting your algorithms.

Features

  • Setup a wallet and deposit funds to your orderly account
  • Check your account balance
  • Check current metrics on current assets
  • Create Positions in various assets
  • Create Limit Orders
  • Create, Backtest and Run a Trading Strategy
